ğŸ§­ Repository Summary (AI)

This Python library provides convenient access to the OpenAI API (v3.8+). It features both synchronous and asynchronous clients, generated from the OpenAI OpenAPI spec using Stainless. Key features include:

  • Text generation using the Responses and Chat Completions APIs.
  • Vision API for image analysis.
  • Realtime API for low-latency conversational experiences.
  • Automatic pagination for list operations.
  • Typed requests/responses, file uploads, webhook verification, and error handling.
  • Configurable retries, timeouts, and HTTP client.
  • Azure OpenAI support via the AzureOpenAI class.
  • Logging capabilities and utilities for accessing raw response data.
  • Detailed documentation and examples are available.
